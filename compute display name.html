Basic Compute Display Name Example

Odoo-তে display_name compute করা হয় যখন তুমি চাও record-এর নাম custom format এ দেখাতে।

সাধারণত _rec_name ব্যবহার করলে একটি field দেখায়, কিন্তু compute display name ব্যবহার করলে একাধিক field combine করে দেখানো যায়।

python model

from odoo import models, fields, api

class HospitalManagement(models.Model):
    _name = 'hospital.management'
    _description = 'Hospital Management'

    name = fields.Char(string="Patient Name")
    age = fields.Integer(string="Age")

    display_name = fields.Char(
        compute="_compute_display_name",
        store=True
    )

    @api.depends('name', 'age')
    def _compute_display_name(self):
        for rec in self:
            rec.display_name = f"{rec.name} ({rec.age})"

Result

যদি data হয়:

Name : John  
Age  : 30

তাহলে display হবে:

John (30)

যদি কেউ এই মডেল কে ইনহেরিট করে তাহলে সেখানে একই ফরম্যাটে ডাটা দেখাবে।
